💜 자바의 정규식

```java
^a : a로 시작하는 단어
[^a] : a가 아닌 철자인 문자 1개
[ab] : a,b중 한 문자
[ab][cd] : a,b중 한 문자와 c,d중 한 문자
[^a-z] : 알파벳 소문자 a부터 z까지를 제외한 모든 문자
[a-z] : 알파벳 소문자 a부터 z까지 하나
[a-z0-9] : 알파벳 소문자 전체,0~9 중 한 문자
```
